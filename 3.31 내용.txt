목록(List) 작성 시
ol ( Ordered List / 순서 o, 마크 o )	하위 : li
ol 속성 : type ( 1, A, I, a, i ), start (시작위치 설정)		il 속성 : value

ul ( Unordered List / 순서 x, 마크 o )	하위 : li
ul 속성 : type ( 모양설정 / none, square, circle, disc )

dl ( Description List / 순서 x, 마크 x )	하위 : dt, dd ( 여러 개 중복가능 )
